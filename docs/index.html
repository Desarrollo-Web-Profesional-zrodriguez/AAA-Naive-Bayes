🔍 Paso 1 — Tokenización: ¿qué palabras reconoce el modelo?

El modelo solo analiza palabras que conoce de su entrenamiento. Las palabras de relleno (el, la, de, y…) se ignoran automáticamente.

✅ Palabras reconocidas y analizadas
⛔ Palabras ignoradas (stop words o fuera del vocabulario)
🗳️ Paso 2 — Votos: cada palabra "vota" por una clase

Para cada palabra reconocida, el modelo consulta la probabilidad aprendida P(palabra | clase). Cuanto mayor el valor, más "pesa" esa palabra para esa clase.

⚖️ Paso 3 — Cálculo final: Prior × Producto de pesos

Naive Bayes multiplica la frecuencia a priori de cada clase por todos los pesos de las palabras. La clase con el producto mayor gana.
